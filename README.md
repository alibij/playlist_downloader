# Music Downloader 🎵

این پروژه یک اسکریپت پایتونی است که نام آهنگ‌ها را از فایل CSV می‌گیرد، آن‌ها را در چند سایت مشخص جستجو می‌کند، لینک دانلود مستقیم (ترجیحاً نسخه 320kbps) را استخراج کرده و فایل‌ها را در پوشه‌ای محلی دانلود می‌کند.  

---

## Features ✨
- جستجو در سایت‌های موسیقی با استفاده از **DuckDuckGo**  
- شناسایی لینک مستقیم MP3 (اولویت با کیفیت 320kbps)  
- دانلود خودکار و ذخیره در پوشه‌ی محلی  
- ثبت لاگ اجرای هر دانلود  

---

## Requirements ⚙️
- Python 3.9+
- کتابخانه‌های مورد نیاز:
```bash
pip install requests beautifulsoup4 duckduckgo-search
